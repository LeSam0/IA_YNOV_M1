# TimeTravel Agency - Webapp Interactive

Webapp pour une agence de voyage temporel fictive, creee avec IA generative.

## 🛠️ Stack Technique
- React + Tailwind CSS
- Framer Motion
- Mistral AI API
- Hebergement : Vercel

## ✨ Features
- Landing page interactive
- Galerie de 3 destinations temporelles
- Chatbot IA conversationnel
- Recommandations personnalisees

## 🤖 IA Utilisees
- Code : Cursor (GPT-5.3 Codex)
- Chatbot : Mistral AI via API Agent
- Visuels : Ressources generees + assets projet
